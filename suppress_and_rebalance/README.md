# Suppress and Rebalance (CVPR 2024)

## Paper
**Suppress and Rebalance: Towards Generalized Multi-Modal Face Anti-Spoofing**  
Conference: CVPR 2024

---

## Problem

Multimodal models typically assume that all modalities provide reliable information.  
However, in real-world scenarios, some modalities contain **noise, domain shift, or artifacts**, which results in **unreliable tokens**.

During multimodal fusion, these unreliable tokens can propagate through **cross-modal attention**, contaminating other modalities and degrading model performance.

Example scenarios:

- Sensor noise in depth images
- Domain shift in RGB images
- Corrupted modality signals

These unreliable tokens can dominate attention weights and negatively influence the fusion process.

---

## Key Idea

The core idea of the paper is:

> Not all tokens should contribute equally during multimodal fusion.

Instead of removing noisy tokens entirely, the method proposes:

1. **Estimating token reliability**
2. **Suppressing unreliable tokens**
3. **Rebalancing the fusion process**

This prevents noisy tokens from contaminating cross-modal interactions while still preserving useful information.

---

## Method Overview

The overall pipeline can be summarized as:

Feature Extraction  
↓  
Token Reliability Estimation  
↓  
Suppression of Unreliable Tokens  
↓  
Rebalanced Cross-Modal Fusion

---

## Token Reliability Estimation

Each token is assigned a reliability score derived from its estimated uncertainty.

Let:

- x_i : token feature  
- u_i : estimated uncertainty  
- r_i : reliability score  

Reliability is inversely related to uncertainty:

r_i = exp(-u_i)

Higher uncertainty results in lower reliability.

---

## Attention Suppression

Given the original attention weights:

A = Softmax(QK^T)

The suppression mechanism modifies the attention weights using the reliability mask:

A' = A ⊙ R

where:

- R : reliability mask  
- ⊙ : element-wise multiplication  

This reduces the influence of unreliable tokens during cross-modal interaction.

---

## Rebalancing Mechanism

After suppressing unreliable tokens, attention contributions are **rebalanced** across tokens to maintain stable training.

Conceptually:

unreliable tokens → reduced influence  
reliable tokens → relatively increased contribution

This ensures the model focuses on reliable information while preserving multimodal interactions.

---

## Relevance to Synthetic Modality Fusion

This repository studies multimodal fusion where one modality may be partially unreliable.

Example scenario:

- **Doppler ultrasound** (relatively reliable modality)
- **Synthetic B-mode generated by RePaint**

Synthetic B-mode may contain:

- copy‑paste artifacts
- inpainting noise
- statistical inconsistencies

However, B-mode still provides useful structural cues:

- lesion contour
- morphology
- spatial structure

Therefore, hard masking is undesirable.

### Mapping to this paper

| My Problem | Paper Concept |
|---|---|
| repaint artifacts | unreliable tokens |
| synthetic B-mode | noisy modality |
| fusion contamination | cross-modal attention noise |
| reliability map | token uncertainty |

Using this framework:

- unreliable patches can be softly suppressed
- structural information can still be preserved

## Key Takeaways

1. Multimodal tokens may have different reliability levels.
2. Noisy tokens can contaminate cross-modal attention.
3. Soft suppression is preferable to hard masking.
4. Reliability-aware fusion improves robustness under noisy modalities.
